# Gene Embedding Evaluation Pipeline

This repository contains a robust, command-line machine learning pipeline designed to benchmark custom gene embeddings (e.g., MiniLM, PubMed embeddings). 

The pipeline evaluates embeddings across two distinct domains:
1. **Binary Classification Tasks (6 Tasks):** Evaluated via 5-Fold Stratified Cross-Validation using ROC-AUC.
2. **Time-Split Prioritization Tasks (2 Tasks):** Positive-Unlabeled (PU) learning evaluated via Median Rank and Fold Enrichment @ 1% (FE_1%). This is specifically designed to simulate real-world disease gene discovery by training on historical targets (pre-2022) and predicting newly discovered targets (post-2022).

## ⚙️ Identifier Translation

* **Classification Tasks:** The Hugging Face classification datasets rely on `Ensembl IDs`. The pipeline automatically uses `mapping/gene_info_table.csv` to translate the Gene Symbols in your custom embedding file into Ensembl IDs.
* **Prioritization Tasks:** The disease gene lists (`DEE` and `CP`) rely on standard `Gene Symbols`. The pipeline handles this seamlessly by maintaining a secondary dictionary of your original, untranslated symbols to perform exact matching for time-split validation.

## 🧠 Model Architectures

For every task, the pipeline evaluates the embeddings using three distinct classifier architectures to capture both linear and complex non-linear feature relationships.

### 1. Logistic Regression (LR)

* **Architecture:** A standard linear model utilizing the logistic function to model class probabilities.
* **Configuration:** Initialized with `class_weight='balanced'` and a maximum of 500 iterations.
* **Role:** Serves as the primary linear baseline. If the embeddings map gene properties into a linearly separable latent space, this model will perform highly.

### 2. Random Forest (RF)

* **Architecture:** An ensemble learning method constructing multiple decision trees during training.
* **Configuration:** Uses `scikit-learn` defaults (100 estimators) with `class_weight='balanced'` and a fixed random state (`2023`) for reproducible splits.
* **Role:** Captures non-linear and hierarchical interactions between specific embedding dimensions without requiring feature scaling.

### 3. Deep Multi-Layer Perceptron (Deep MLP)

* **Architecture:** A deep feedforward artificial neural network.
* **Configuration:** * **Topology:** 10 Hidden Layers `(512, 256, 256, 128, 128, 64, 64, 32, 32, 16)`.
* **Optimization:** Adam optimizer with a maximum of 500 iterations.
* **Preprocessing:** Embedding inputs are explicitly passed through a `StandardScaler` (zero mean, unit variance) prior to training to ensure rapid and stable gradient descent.


* **Role:** A deep learning approach ideal for teasing out highly complex, synergistic patterns in the embedding vectors, particularly for genome-wide disease gene prioritization.

## 🚀 Usage

Execute the pipeline via bash (or queue it on an HPC cluster like Spartan). You only need to provide the path to your embeddings and a unique tag for the model run.

```bash
python run_genept_eval.py \
    --embeddings ./embeddings/gene_embeddings_all_minilm_l6_v2.csv \
    --tag "MiniLM_v2"

```

### Optional Arguments

* `--out_dir`: Specify a custom directory to save the outputs (Default: `./outputs`).

## 📊 Outputs

Upon completion, the script prints a formatted summary table to your terminal and generates four files in your output directory:

### 1. Classification Results

* `classification_results_<tag>.csv`: Raw ROC-AUC scores combined with the loaded baselines.
* `classification_plot_<tag>.png`: A grid plot comparing your embeddings against the baseline across all 6 classification tasks.

### 2. Prioritization Results

* `prioritization_results_<tag>.csv`: Raw Median Rank and Fold Enrichment @ 1% scores.
* `prioritization_plot_<tag>.png`: A side-by-side grid plot comparing your embeddings against the Baseline. The top row visualizes Median Rank (lower is better), and the bottom row visualizes Fold Enrichment (higher is better).
